BMore Local - Event Retweets
Events You Should Have Heard About

What?

Follow us on Twitter to hear about events that you should have heard about

Know a maven? Get them using the #bmoevent hashtag to announce an event. We'll pick them up.

The purpose of BmoreLocal is to make it easy to find interesting events by leveraging Twitter. We have selected tweeters to be Event Mavens, which means every time they tweet about an event and use the relevant hash tag, we retweet them @bmorelocal. That's it.

We plan on starting with events and then looking at topic related feeds at some point soon. Our code is on github: awgrover/curated-hash-feed . The first version was built during the 2010 Baltimore Hackathon

Why?

Over a beer one night, we realized it was time to stop not hearing about all of the cool events that happen in Baltimore like trapeze artistry in Station North, food-making meet-ups, and taco crawls. Given how busy everyone is, we thought that it would be great to be able to quickly scan tweets related to interesting events.

How to Help

We are looking for moderators, event mavens, and people to spread the word about this project and give us feedback. You can work on an existing hash tag, or have us start a new one. Tweet at us to get involved, or start using the hash tag, and we will look into adding you as a maven.

Who?

Alan and Heather started this, but this is a group effort. We have Moderators, people who have offered to take a lead in finding event mavens and getting them involved, and the event mavens themselves.
